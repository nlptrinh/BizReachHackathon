# BizReachHackathon

## How to run:
First, set your GitHub account in credentials.json (for data crawling).

```
pip install -r requirements.txt
export FLASK_APP=website.py
python -m flask run
```

PowerShell (Windows):
```
$env:FLASK_APP = "website.py"
python -m flask run
```
